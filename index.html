O que é Git, Github e modularização ?

Git é um projeto open source desenvolvido pelo criador do Linux, Linus Torvalds. Já o Github é um sistema desenvolvido para salvar e versionar codigo, que foi desenvolvido utilizando a ferramenta Git e por fim foi comprado pela Microsoft. E Modularização é um conceito computacional que é empregado para dividir o seu programa em partes funcionais, partes essas que conversam umas com as outras

Versão do sistema(1.0,2.0) é o mesmo que versionar o codigo?

Versão do sistema é o número de versões que o projeto passou ate chegar em uma versão final para publicação, onde cada número representa uma mudança. Como o 1º número representa uma grande mudança no sistema o 2º uma mudança de funcionalidade e o 3º número uma mudança pequena, por exemplo: o cabeçalho de um relatorio gerado pelo programa. Versionar é para documentar inclusões, alterações e exclusões de funcionalidades, mas não necessariamente chegará ao público final.
